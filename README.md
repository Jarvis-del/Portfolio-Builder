A lightweight, browser-based portfolio builder that lets you generate a polished personal portfolio page in minutes — no coding, no backend, no sign-up.

✨ Features
1. 5 professionally designed templates — Folio, Console, Studio, Atelier, and Signal — each with a distinct visual style (editorial serif, dev-console dark mode, bold geometric, minimal, and gradient-accented dark UI)
2. Simple 3-step flow — pick a template → fill in your details → preview your generated portfolio
3. Privacy-first by design — no phone, email, or LinkedIn fields; only professional details are shown on the generated page
4. Local resume upload — attach a PDF resume that's handled entirely in-browser via the File API and never uploaded to any server
5. Fully client-side — built with plain HTML, CSS, and JavaScript, no frameworks, no build step, no dependencies
5. Live preview — instantly see your portfolio rendered in the chosen template, with the ability to go back and edit details or switch              templates anytime


##  Tech Stack
HTML5
CSS3
Vanilla JavaScript (ES6+)
Google Fonts (Fraunces, Inter, Space Grotesk, JetBrains Mono, Sora)

## Getting Started
No installation or build tools required.
Clone the repository

bash   git clone https://github.com/your-username/portfolio-forge.git

Open index.html in your browser

That's it — the app runs entirely client-side.
📌 How It Works

Choose a template from the gallery on the landing screen
Fill in your details — name, title, bio, skills, up to two projects, experience, education, and optionally a resume PDF
Generate your portfolio — view a fully rendered, styled preview built from your inputs, with the option to edit details or change templates at any time
